# OpenVINO_car_detection
The showcase to perform synchronous inference on an image to check whether a car is existed, and then draw the bounding box to where it located at.

| Details of Software |                    |                   
|---------------------|--------------------|
| OS:                 | Ubuntu\* 18.04 LTS |
| OpenVINO:           | 2020.3 LTS         |
| Python:             |  3.6.9             |
